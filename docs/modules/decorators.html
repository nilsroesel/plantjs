Endpoint

  • Endpoint(config: EndpointConfig): (Anonymous function)
    • Defined in decorators/endpoint.decorator.ts:38

    Property Decorator Function

    Creates an endpoint of a function The decorated function need the following index signature otherwise it will throw an error (request: Request, response: Response, next?: Function) (also an indexsignature fore middlewares) If you pass the next-function and call it, it will get the next function from the route middlewares and abort the current execution

    example
    
  @Component({route: '/api'})
  class Test {

     @Endpoint({
         route: '/foo'
         middleware: [(request: request) => request.params['foo'] = foo; ]
     )
     foo(request: Request, response: Response) {
         response
          .status(200)
          .json(request.params.foo)
          .send();
     }
  }

    Parameters

    • config: EndpointConfig

    Returns (Anonymous function)
